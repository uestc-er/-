猫分类器 - 逻辑回归实现
这是一个使用逻辑回归算法实现的猫/非猫图像分类器。

项目概述
该项目使用深度学习中的逻辑回归模型来识别图像中是否包含猫。代码实现了从数据加载、预处理、模型训练到预测评估的完整流程。

功能特性
数据预处理：图像扁平化、归一化处理

模型实现：从零实现逻辑回归算法

训练优化：使用梯度下降法优化参数

性能评估：输出训练集和测试集准确率

可视化：损失曲线和预测结果展示

核心函数说明
1. 数据加载 (load_dataset)
从H5文件中加载训练集和测试集

返回图像数据、标签和类别信息

2. 辅助函数
sigmoid(): 激活函数

initialize_with_zeros(): 参数初始化

propagate(): 前向和反向传播

optimize(): 梯度下降优化

3. 预测函数
predict(): 使用训练好的模型进行预测

model(): 整合所有功能的完整模型

参数配置
在 model() 函数中可以调整以下超参数：

num_iterations: 迭代次数（默认2000）

learning_rate: 学习率（默认0.5）

print_cost: 是否打印训练过程中的损失值

注意事项
需要提前下载数据集文件并修改 load_dataset() 中的文件路径

图像尺寸为64×64像素，3通道RGB

数据集包含209个训练样本和50个测试样本

模型性能
该简单逻辑回归模型在猫分类任务上通常能达到约70%的测试准确率

训练准确率可接近100%，存在一定过拟合

可通过调整超参数或使用更复杂模型进一步提升性能
